Disclaimer!: This program is functional but is still being worked on!

The intention of this program is to help car salesmen and dealerships, but it can
also be used as a price comparison tool for those shopping for a car.

If you think you can improve this project, please don't hesitate to commit a change to the code!

Pandas and Selenium are both requred for this program. These are both included in the requirements.txt.
To install the requirements.txt, type in your terminal "pip install -r requirements.txt"

Pandas documentation: https://pandas.pydata.org/docs/user_guide/index.html

Selenium:
  -Downloading Chromedriver: https://chromedriver.chromium.org/downloads
  -Documentation: https://selenium-python.readthedocs.io/
